# Votos por Localidade - Eleições 2022

UI para pesquisar as porcentagens de votos dos candidatos classificados para o segundo turno por localidade

## Como executar

- Instale as dependências:
 
```
 pip install -r requirements.txt
```
 
 - Execute

```
python index.py
```
